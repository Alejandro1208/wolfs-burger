Wolf's Burger - Website Template

1. Introduction

Thank you for purchasing the Wolf's Burger website template!

This is a complete and self-manageable website solution, perfect for restaurants, cafes, and food businesses. It features a modern frontend built with React (Vite + TailwindCSS) and a powerful backend powered by PHP and MySQL, allowing you to manage your entire content from a secure admin panel.

Key Features:

  • Full Admin Dashboard to manage the site.
  • Product and Category Management.
  • Drag & Drop to reorder categories.
  • Easy customization of website colors.
  • Full control over homepage content, sliders, and buttons.
  • General settings management (logo, contact info, social media, etc.).

2. Server Requirements

To run this project, your server must meet the following requirements:

  • PHP Version: 8.0 or higher.
  • Database: MySQL or MariaDB.
  • PHP Extensions: mysqli, json.
  • Web Server: Apache or Nginx with mod_rewrite enabled for URL rewriting.

3. Installation Guide

Please follow these steps carefully to install the website on your server.

Step 1: Upload Files

Unzip the main package you downloaded. Inside, you will find the folders api and dist. Upload the contents of both folders to the public directory of your server (e.g., public_html or www).

Step 2: Create a Database

Using your hosting control panel (like cPanel), create a new database. Then, create a new database user and add that user to the database, granting it All Privileges. Make sure to save the database name, username, and password.

Step 3: Import the Database

In your hosting control panel, find phpMyAdmin and select your new database. Click on the "Import" tab. Upload the database.sql file included in the package and click "Go". This will create all the necessary tables with default data.

Step 4: Configure the Database Connection

On your server, navigate to the api folder and open the file db_connection.php with a text editor.

Edit the following lines with your database credentials from Step 2:

<?php
define('DB_HOST', 'localhost');
define('DB_USER', 'your_database_user');     // <-- CHANGE THIS
define('DB_PASS', 'your_database_password');  // <-- CHANGE THIS
define('DB_NAME', 'your_database_name');      // <-- CHANGE THIS
...
?>

Step 5: Configure the API URL in the Frontend

CRITICAL STEP: This step connects the visual part of the site with your database. Do not skip it.

Open the file dist/assets/index-....js (the name will have random characters). You must edit this file.

Search for the text https://alejandrosabater.com.ar/api.

Replace it with the public URL to your api folder. For example: https://yourdomain.com/api.

Step 6: Access Your Site

That's it! Your website should now be live.

To log into the admin panel, go to https://yourdomain.com/admin/login.

Default login credentials are:

  • Email: admin@wolfs.com
  • Password: admin123
Important: Please change these credentials immediately after your first login via the "Usuarios" (Users) section!

4. Admin Panel Usage

The admin panel is designed to be intuitive. Here is a brief overview of each section:

  • Dashboard: A summary of your site's content.
  • Portada (Homepage): Manage the background image carousel and the main hero text and buttons.
  • Categorías (Categories): Create and manage your product categories. You can drag and drop them to change their display order on the site.
  • Productos (Products): Manage all your products, including names, descriptions, prices, images, and categories.
  • Apariencia (Appearance): Change the main colors of your website to match your brand.
  • Configuración (Settings): Update your site's logo, contact details, social media links, Google Maps URL, and footer text.
  • Usuarios (Users): (Superadmin only) Manage admin users who can access the panel.
